Om projektet

C# Numerics är ett lättanvänt bibliotek med extension-metoder och datatyper för komplexa tal, vektorer, matriser, differentialekvationer, Fourier/Laplace-transformer, statistik och mycket mer.

Höjdpunkter

Numeriska extension-metoder

Factorial, Newton–Raphson, derivator, integraler, Monte Carlo.

Linjära algebra

Matrix, Vector, determinant, inverse, eigenvalues och linjära system.

Vektorfält

Gradient, divergence, curl, laplacian och export till CSV.

Spectral & transform

FFT, DFT, Laplace-transform och signalbehandling.

Kodexempel

Factorial

// 5! = 120
int result = 5.Factorial();

Newton–Raphson (exempel)

Func f = x => Math.Pow(x,2) - 4;
double root = f.NewtonRaphson(); // 2

Runge–Kutta (RK4)

Func<(double t, double y), double> ode = v => Math.Tan(v.y) + 1;
var result = ode.RungeKutta(1, 1.1, 0.025, 1);

Fler exempel finns i README och i repository: kodexempel, enhetstester och notebooks för visualisering.
