grouped_timeserie_cv

Bibliotek för gruppad tidsserie-cross-validation i Python.

Install via PyPI

Om projektet

grouped_timeserie_cv är ett Python-bibliotek som bygger på scikit-learn, med stöd för **grupperad tidsserie-cross-validation** för klassificering och regressionsmodeller.

Höjdpunkter

Gruppad cv

Korsvalidering där varje grupp är tidssekvens och modeller tränas på resterande data.

Klassificering & regression

Stöd för klassificering och regressionsmetoder.

Pipeline & hyperparameter-sökning

Integreras med scikit-learn-pipelines och grid-search

Visualisering & resultatanalys

Plot-verktyg för confusion matrix, learning curve, prediktion vs verkligt.

Kodexempel

Installation

pip install grouped-timeserie-cv

Användning (klassificering)

from grouped_timeserie_cv import GroupedTimeSerieCV

grouped_cv = GroupedTimeSerieCV()
result = grouped_cv.classify(
    data, pipelines, param_grids,
    frequency='D', datetime_col='DateTime',
    label_col='Label', scoring='accuracy'
)

Användning (regression)

result = grouped_cv.predict(
    data, pipelines, param_grids,
    frequency='D', datetime_col='DateTime',
    label_col='Label', scoring='neg_mean_squared_error'
)
